## Triskly とは？

Trisklyは「Three Wins」メソッドを実装したシンプルな目標管理アプリです。

- 今日の3つの目標
- 今週の3つの目標
- 今月の3つの目標

それぞれのタイムフレームで3つの重要な目標に集中することで、生産性を高めます。

## サポートプラットフォーム

- macOS (Apple Silicon & Intel)
- Windows (x64)
- Linux (Debian/Ubuntu, AppImage)
